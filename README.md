🥿 ООО «Обувь» — Система управления складом и заказами
Приложение на Python (Tkinter) для автоматизации работы магазина обуви: учёт товаров, управление заказами, пункты выдачи, роли пользователей.

📋 Требования
Python 3.10+ (протестировано на Python 3.14)

Библиотеки:
pip install pandas openpyxl Pillow

🚀 Первый запуск (пошагово)
Шаг 1. Установите зависимости
pip install pandas openpyxl Pillow
Или:

Шаг 2. Создайте базу данных
Запустите скрипт создания БД:

python scripts/create_database.py
Что произойдёт:

✅ Создастся папка database/ (если нет)

✅ Создастся файл database/obuv.db

✅ Создадутся все таблицы по скрипту database.sql


Шаг 3. Импортируйте данные из Excel
Запустите скрипт импорта:

bash
python scripts/import_data.py
Что произойдёт:

✅ Очистятся существующие таблицы (кроме пользователей)

✅ Импортируются пользователи, товары, пункты выдачи, заказы


Шаг 4. Запустите приложение
Теперь запустите основное приложение:

bash
python app/main.py
Откроется окно авторизации.

Шаг 5. Войдите в систему

▶️ Повторный запуск приложения
Если база данных уже создана и данные импортированы, просто запустите:

bash
python app/main.py
Приложение откроется мгновенно.

🔄 Как обновить данные из Excel (полный сброс)
Если вы изменили файлы .xlsx и хотите обновить данные в БД:

Вариант 1: Полный сброс (рекомендуется)
Удалите базу данных:

bash
# Windows (PowerShell)
Remove-Item database\obuv.db

# Windows (cmd)
del database\obuv.db

# macOS / Linux
rm database/obuv.db
Создайте БД заново:

bash
python scripts/create_database.py
Импортируйте данные:

bash
python scripts/import_data.py
Запустите приложение:

bash
python app/main.py

Вариант 2: Только перезапуск импорта (без пересоздания БД)
Если структура БД не менялась, можно запустить только импорт (данные очистятся и импортируются заново):

bash
python scripts/import_data.py
⚠️ Внимание: это очистит все таблицы (кроме пользователей) и импортирует данные заново из Excel.
