## 🚀 Первый запуск (пошагово)

### Шаг 1. Установите зависимости

```bash
pip install pandas openpyxl Pillow
```

### Шаг 2. Создайте базу данных

Запустите скрипт создания БД:

```bash
python scripts/create_database.py
```

**Что произойдёт:**
- ✅ Создастся папка `database/` (если нет)
- ✅ Создастся файл `database/obuv.db`
- ✅ Создадутся все таблицы по скрипту `database.sql`

### Шаг 3. Импортируйте данные из Excel

Запустите скрипт импорта:

```bash
python scripts/import_data.py
```

**Что произойдёт:**
- ✅ Очистятся существующие таблицы (кроме пользователей)
- ✅ Импортируются пользователи, товары, пункты выдачи, заказы

### Шаг 4. Запустите приложение

```bash
python app/main.py
```

Откроется окно авторизации.

## 🔄 Как обновить данные из Excel (полный сброс)

### Вариант 1: Полный сброс (рекомендуется)

1. **Удалите базу данных:**

   ```bash
   # Windows (PowerShell)
   Remove-Item database\obuv.db

   # Windows (cmd)
   del database\obuv.db

   # macOS / Linux
   rm database/obuv.db
   ```

2. **Создайте БД заново:**
   ```bash
   python scripts/create_database.py
   ```

3. **Импортируйте данные:**
   ```bash
   python scripts/import_data.py
   ```

4. **Запустите приложение:**
   ```bash
   python app/main.py
   ```

### Вариант 2: Только перезапуск импорта (без пересоздания БД)

Если структура БД не менялась:

```bash
python scripts/import_data.py
```

> ⚠️ **Внимание:** это очистит все таблицы (кроме пользователей) и импортирует данные заново из Excel.
